部署步骤总结
第 1 步：打包发布包
在你的电脑上打开 PowerShell，进入项目目录：

cd e:\Files\pixel-lab
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\deploy\package-release.ps1

这会生成：deploy\build\pixel-lab-release.zip

第 2 步：上传到服务器

scp .\deploy\build\pixel-lab-release.zip root@118.31.23.209:/tmp/

第 3 步：SSH 登录服务器

ssh root@118.31.23.209

第 4 步：解压

rm -rf /tmp/pixel-lab-release
mkdir -p /tmp/pixel-lab-release
unzip -o /tmp/pixel-lab-release.zip -d /tmp/pixel-lab-release
cd /tmp/pixel-lab-release

第 5 步：部署应用

sudo bash server/deploy-war.sh

看到类似这样的输出就是成功：
{"code":200,"msg":"success","data":{"status":"ok","database":"ok"}}

⚠️ 注意事项
阿里云防火墙：确保放行了 22 和 80 端口
数据库密码：不要用简单密码，且不能包含单引号 ' 和反斜杠 \
API Key：DeepSeek 或通义千问的 Key 不能是默认的 CHANGE_ME 值

怎么查看/etc/pixel-lab/pixel-lab.env的内容

cat /etc/pixel-lab/pixel-lab.env
